Quark

20 January 2022

My current game and game engine project.

What is it

Quark is the name of a game engine

I personally believe it's quite challening to develop a game engine without a game also in the works, as it makes it hard to develop the right tools for the game engine if you have no idea what kinds of games are going to be made with it.

My plans for the game are a fast paced first person shooter I'm not aiming for any strict realism with the graphics, I more want something that is unique and colorful.
